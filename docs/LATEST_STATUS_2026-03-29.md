# 最新进展（2026-03-29）

本文档用于说明当前毕业设计的真实进展，并明确区分：

- 当前论文主方案
- 前期尝试与对照路线
- 当前已完成到什么程度

## 1. 当前主方案是什么

当前主方案已经从前期多模型探索，转入围绕开题报告问题的针对性改进路线。

当前主方案为：

- `MicroNet-pretrained SE-ResNeXt50 + U-Net++ + Edge Head + Deep Supervision`
- 在统计约束版本中加入 `VF Loss`

这条路线对应研究区：

- [`docs/mbu_netpp_research/README.md`](mbu_netpp_research/README.md)

之所以切到这条线，是因为开题报告里明确提出了以下核心问题：

- `SEM` 图像中 γ/γ′ 相灰度差异小
- 颗粒边界复杂，容易粘连或漏检
- 高质量标注样本少
- 课题最终目标包含面积分数和尺寸统计，要求结果稳定

因此，当前主方案不是简单换模型，而是针对这些问题做的结构化改进。

## 2. 当前主要数据来源

当前主监督数据不是大规模公开数据，也不是零样本结果直接拿来训练，而是：

- 前期使用 `MatSAM` 生成粗标注
- 再人工逐张精修
- 最终形成 `9` 张高质量标注图

这 `9` 张精修图是当前阶段最核心的金标准监督数据，也是当前 `E1a-E5` 全部实验的基础。

当前补充数据与辅助数据包括：

- `dataset/full_png/` 中的 `100` 张原始 `PNG`
- 修正信息栏后的 [`dataset/full_png_cropped_xlsx/`](../dataset/full_png_cropped_xlsx/)
- NASA `Super1~4` 外部验证数据

但这些内容的定位不同：

- `9` 张精修图：当前主监督数据
- `full_png`：未标注池、筛难例和后续增量训练候选
- NASA 数据：外部验证与泛化参考，不是同分布替代数据

## 3. 前期路线现在如何定位

仓库里保留的前期路线仍然有价值，但现在都不再作为当前论文主线。

### 3.1 `SAM LoRA V2`

- 前期高精度边界尝试
- 现在作为历史比较材料保留

### 3.2 `ResNeXt50`

- 前期统计型尝试
- 现在作为历史比较材料保留

### 3.3 `MatSAM`

- 零样本对照方法
- 现在主要作为对照和粗标注起点

### 3.4 `MatSAM + SAM2 + localfix_strict_otsu`

- 自动预标注路线
- 现在主要用于生成粗底稿、支持人工精修和扩标

一句话说清楚：

> 这些前期路线现在的价值是“提供对照背景和扩标入口”，不是“当前论文最终主方案”。

## 4. 当前实验进度

当前主方案的核心实验主线已经完成到：

- `E1a`：数据增强对照
- `E1`：`U-Net / U-Net++ / MicroNet-U-Net++` 基线
- `E2`：`Edge Head`、`Edge Head + Deep Supervision`
- `E3`：在 `E2` 上加入 `VF Loss`
- `E4`：半监督伪标签验证
- `E5`：后处理验证

### 4.1 当前最强分割/边界版本

- `E2 = MicroNet-U-Net++ + Edge Head + Deep Supervision`
- `Dice = 0.9321`
- `Boundary F1 = 0.7276`
- `VF = 0.0371`

### 4.2 当前更均衡的统计版本

- `E3 = E2 + VF Loss`
- `Dice = 0.9309`
- `Boundary F1 = 0.7211`
- `VF = 0.0332`

### 4.3 `E4` 的真实结论

当前 `E4` 是务实版半监督验证：

- teacher 使用 `E2`
- 从 `full_png` 中剔除与 `9` 张精标图重名的图后，实际用了 `91` 张未标注图
- 伪标签为 `0.5` 阈值硬伪标签

结果：

- `Dice +0.0020`
- `Boundary F1 +0.0105`
- `VF` 误差变差 `+0.0021`

结论：

- `E4` 对分割和边界有小幅正收益
- 但对统计没有帮助，暂时不能直接升级为最终方案

### 4.4 `E5` 的真实结论

`E5` 的主口径结果表明：

- `remove_small` 只有极小收益，可以保留为轻量微调
- 激进的形态学平滑会明显打坏结果，应明确否定

因此：

- 后处理不是当前方案的主要增益来源
- 继续花大量时间堆后处理规则，收益不会高
