## 1. IN718Net 深挖结论

### 1.1 已确认的事实

- `IN718Net` 来自论文  
  [Computer Vision Approaches for Segmentation of Nanoscale Precipitates in Nickel-Based Superalloy IN718](https://link.springer.com/article/10.1007/s40192-020-00195-z)
- 论文明确写到：
  - 数据来自 NASA Glenn Research Center
  - 共 `47` 张 `900 x 900 px` SEM 图
  - 三类标签：`γ″ / γ′ / matrix`
  - 通过翻转扩增到 `188` 张
- 论文同时明确写到：**可下载的是代码**，不是数据集本体。文中只说：
  - 复现实验所需代码可从 GitHub 下载
  - 或向通讯作者索取

### 2.3 二值化映射

NASA 官方 notebook 中，`Super1~4` 的多类标注定义为：

- `secondary = [255, 0, 0]`
- `tertiary = [0, 0, 255]`
- `matrix = 其余区域`

针对当前课题的二分类目标 `γ′ vs matrix`，本轮外部验证采用：

- `foreground = secondary + tertiary`
- `background = matrix`

这是一个有意的任务对齐简化。它适合做“外部泛化检查”，但不等于和你本地 `γ/γ′` 任务严格同分布。

## 3. NASA 外部验证结果

### 3.1 评估口径

本轮外测有两种口径：

- 补充口径：`standard52`  
  把 `train + val + test` 全部当作外部已标注集看泛化

主口径更值得写进报告，因为它更接近“按官方测试集做外部验证”。

### 3.2 主口径：官方 `test + different_test`

| 模型 | overall Dice | overall VF | overall Boundary F1 | test Dice | different_test Dice |
|---|---:|---:|---:|---:|---:|
| U-Net++ | 0.4822 | 0.2726 | 0.3516 | 0.6057 | 0.2723 |
| MicroNet-U-Net++ | 0.5565 | 0.2369 | 0.4334 | 0.6975 | 0.3169 |
| MicroNet-U-Net++ + Edge + Deep | 0.4494 | 0.2807 | 0.3353 | 0.5670 | 0.2494 |
| MicroNet-U-Net++ + Edge + Deep + VF | **0.5663** | **0.2302** | **0.4441** | 0.6890 | **0.3577** |

### 3.3 结果解读

- `MicroNet-U-Net++` 比普通 `U-Net++` **明显更能泛化到 NASA**。
- `Edge + Deep` 在你自己的 `9` 张监督集上是最好的分割结构，但到了 NASA 外域后，**反而退化最明显**。
- `Edge + Deep + VF Loss` 在外域整体最好：
  - `overall Dice / VF / Boundary F1` 都是 4 个候选里最优
  - 在更难的 `different_test` 上也最好
- 但如果只看稳定性，`MicroNet-U-Net++` baseline 更稳：
  - `test Dice` 三折标准差约 `0.0307`
  - `Edge + Deep + VF` 的 `test Dice` 三折标准差约 `0.1108`

### 3.4 对主模型选择的影响

基于 NASA 外测，当前更合理的结论是：

- **本地交叉验证最强分割结构**：`MicroNet-U-Net++ + Edge + Deep`
- **跨域更均衡的最终候选**：`MicroNet-U-Net++ + Edge + Deep + VF Loss`
- **最稳的外部泛化 baseline**：`MicroNet-U-Net++`

因此，如果论文/答辩要兼顾“本地性能 + 统计任务 + 外部验证”，当前最保守、最能自圆其说的主模型还是：

- `MicroNet-U-Net++ + Edge Head + Deep Supervision + VF Loss`

## 4. 同类公开数据集梳理

### 4.1 可直接下载

#### A. NASA Super1~4

- 链接：[nasa/pretrained-microscopy-models](https://github.com/nasa/pretrained-microscopy-models)
- 类型：Ni-based superalloy 显微分割 benchmark
- 标签：secondary / tertiary / matrix
- 匹配度：**最高**
- 可直接用于：
  - 外部验证
  - 继续预训练/域适应
  - 迁移对比实验

#### B. Ni-WC MMC SEM segmentation dataset

- 链接：[Zenodo 10.5281/zenodo.17315241](https://zenodo.org/records/17315241)
- 类型：Ni-WC 金属基复合材料 SEM 语义分割
- 规模：`405` 个 `512 x 512` 图块和对应 mask
- 匹配度：**中等**
- 说明：
  - 是镍基、SEM、像素级分割
  - 但任务是 MMC 组织分割，不是 γ/γ′ 析出相

#### C. LPBF / IN718 原始 SEM 数据集

- 链接：[Zenodo 10.5281/zenodo.14163786](https://zenodo.org/records/14163786)
- 类型：IN718 原始显微图数据
- 内容：LM、SEM、EDS、力学测试等
- 匹配度：**低到中等**
- 说明：
  - 可做无监督预训练或风格参考
  - **没有像素级分割标签**

#### D. Inconel 718 cryogenic deformation raw SEM dataset

- 链接：[Dryad 10.5061/dryad.jm63xsjk3](https://datadryad.org/dataset/doi%3A10.5061/dryad.jm63xsjk3)
- 类型：IN718 原始高分辨 SEM 图
- 内容：多步变形前后原始图像
- 匹配度：**低到中等**
- 说明：
  - 可做预训练、表观域分析或自监督
  - **没有分割标注**

## 5. 当前建议

### 5.1 数据策略

- 本地 `9` 张精修图仍然是你当前最核心的金标准监督数据
- NASA `Super1~4` 更适合作为：
  - 外部验证集
  - 迁移/域适应对照
  - 额外公开 benchmark
- 不建议直接把 NASA 和你自己的数据无脑混训，因为两者形貌差异明显

### 5.2 论文写法

这一轮可以比较诚实地写成：

- `IN718Net` 是文献中与本课题最接近的数据集之一，但目前未找到公开下载入口，因此无法直接纳入复现实验。
- NASA `Super1~4` 是当前可直接获取、任务最接近的公开镍基超合金分割数据，因此被引入作为外部泛化验证集。
- 外部验证表明，`MicroNet` 预训练确实增强了跨域泛化，而 `Edge + Deep` 这类针对本地边界恢复的结构改造在跨域场景下不一定稳定成立。
