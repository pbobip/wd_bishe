## 一、主任务口径

论文任务建议表述为：

```text
面向镍基单晶 SEM 图像的 gamma_prime 相二值语义分割与组织统计分析
```

任务类型为像素级二值语义分割。模型输入为单通道灰度 SEM 图像，输出为目标相区域二值 mask。系统中进一步基于 mask 生成 overlay、体积分数、颗粒数量、面积、尺寸和通道宽度等统计结果。

| 口径 | 数量 | 是否增加物理文件 | 论文建议写法 |
| --- | ---: | --- | --- |
| 原始监督图像 | 53 张 | 是，真实图像文件 | 本文主数据集包含 53 张 SEM 图像 |

| 每 fold 验证图像 | 9/8 张 | 否，交叉验证划分 | fold0-2 为 9 张验证，fold3-4 为 8 张验证 |
| 每 epoch 增强训练图像块 | 192 张 | 否，在线随机采样 | 每轮训练从训练图像中随机裁剪并增强 192 张 256x256 图像块 |
| 每 fold 增强训练图像块 | 15360 张 | 否，在线增强样本 | 192 张/epoch x 80 epoch |
| 5 折增强训练图像块合计 | 76800 张 | 否，重复训练采样量 | 5 折训练过程中优化器看到的 256x256 增强图像块总量 |
| 3 seed 重复实验增强训练图像块合计 | 230400 张 | 否，重复训练采样量 | 76800 张/run x 3 seeds |
| 离线等价 image/mask/edge 文件数 | 691200 个 | 仅模拟落盘时才生成 | 230400 张图像块 x image/mask/edge 三件套 |

数据增强口径：增强为在线增强，不生成固定扩增图片文件夹。因此不能写成“增强后得到 76800 张完整 SEM 原图”，应写成“5 折训练过程中累计形成 76800 张 256x256 增强训练图像块”。裁剪和交叉验证本身不创造新的独立原始图像；真正增加训练多样性的是在线随机裁剪与随机增强。

按当前代码计算，数据增强不是离线复制完整图像，而是在每次读取训练样本时对 256x256 图像块进行随机增强。因此当前代码没有“增强后完整 SEM 图像数量”，只有“增强后训练图像块数量”。不要写 8 倍增强，也不要写 424 张增强完整图像；这不是当前配置真实执行方式。

当前增强配置及 5 折训练过程中的期望触发次数如下：

| 增强操作 | 概率 | 单次 5 折训练期望次数 | 3 seed 期望次数 |
| --- | ---: | ---: | ---: |
| 增强流水线输入图像块 | 1.00 | 76800 | 230400 |
| 水平翻转 | 0.50 | 38400 | 115200 |
| 垂直翻转 | 0.50 | 38400 | 115200 |
| RandomRotate90 被调用 | 0.75 | 57600 | 172800 |
| RandomRotate90 产生非 0 度旋转 | 0.5625 | 43200 | 129600 |
| 缩放仿射 | 0.30 | 23040 | 69120 |
| 亮度/对比度扰动 | 0.25 | 19200 | 57600 |
| 高斯噪声 | 0.20 | 15360 | 46080 |

说明：`RandomRotate90(p=0.75)` 被调用后会在 0、90、180、270 度中随机选择，因此真正产生非 0 度旋转的期望概率为 `0.75 x 3/4 = 0.5625`。缩放、亮度/对比度和高斯噪声属于连续随机扰动，理论上可产生大量不同图像，但代码不会提前枚举或保存这些图像。

论文推荐表述：

```text
本文原始监督数据包含 53 张 SEM 图像。预处理阶段裁切图像底部信息栏后，完整图像数量仍为 53 张。模型训练阶段采用在线随机裁剪和数据增强策略，将图像裁剪为 256x256 的训练图像块。根据训练配置，每个 fold 每个 epoch 采样 192 张增强训练图像块，训练 80 个 epoch，因此每个 fold 累计形成 15360 张训练图像块；5 折交叉验证累计形成 76800 张训练图像块。需要说明的是，这些图像块由在线增强动态生成，不作为固定图片文件单独保存。
```

## 八、论文结论口径

推荐结论写法：

1. 传统阈值和边缘方法难以稳定处理 SEM 相区域分割，尤其在边界连续性和体积分数误差上明显弱于深度模型。
2. MBU-Net++ 通过 U-Net++ 多尺度特征融合、MicroNet 显微图像预训练、边界辅助监督和深监督，提高了小样本 SEM 图像分割效果。
3. 边界采样是最终模型的重要优化策略，使 holdout Boundary F1 从 baseline 的 0.9192 提升到 0.9300。
4. NASA 相似数据仅加入训练时可以略微提升 Dice，但边界指标不如主模型，因此更适合作为辅助训练扩展实验。
5. 当前实验可以支撑论文模型有效性，但由于主数据集规模较小，应客观说明泛化能力仍需更多外部测试验证。
