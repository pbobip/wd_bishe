# 毕业设计整理目录

本目录是论文写作和答辩准备用的整理版归档。原始仓库 `D:\中期\wd_bishe` 未被移动或破坏，本目录主要复制论文需要的源码、训练代码、实验结果、数据清单和说明材料。

## 目录结构

| 目录 | 内容 |
| --- | --- |
| `00_说明文档` | 总说明、实验结果索引、仓库说明文档 |
| `01_系统实现` | 前端、后端和系统启动脚本，不包含 `node_modules`、运行缓存和后端历史运行数据 |
| `02_模型训练与实验结果` | `MBU-Net++` 训练代码、配置、权重、数据划分清单、两轮主实验结果和最终推理结果 |
| `03_其他资料` | 数据集、外部数据、样例、根目录杂项和模块资料 |
| `04_论文资料` | 开题报告、论文模板、答辩材料和参考资料 |

## 使用顺序

1. 先看 `00_说明文档/目录说明.md`，了解每个文件夹放了什么。
2. 再看 `00_说明文档/实验结果索引.md`，确定论文表格用哪些结果。
3. 写系统章节时看 `01_系统实现`。
4. 写模型和实验章节时看 `02_模型训练与实验结果`。
5. 写论文格式、开题和答辩材料时看 `04_论文资料`。
